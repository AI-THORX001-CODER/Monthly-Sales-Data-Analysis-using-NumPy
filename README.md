📊 Monthly Sales Data Analysis using NumPy

A simple Python project that analyzes monthly sales data using the NumPy library. The program calculates average sales, identifies the best-performing month, finds the highest sales value, and measures overall sales growth.

🚀 Features
Store monthly sales data in NumPy arrays
Calculate average monthly sales
Identify the best sales month
Find highest sales value
Calculate sales growth percentage
Display sales performance summary

Python 3
